# Проект "Dogs Blog"

### Особенности проекта:
- Регистрация/Авторизация пользователя
- Личный кабинет пользователя
- Доступ к DummyApi
- Пагинация страниц
- Ant Design

### Быстрый старт

- Клонирование репозитория
```
git clone https://github.com/vladimirkrylov01/Dogs_Blog.git
```
- Установка зависимостей
```
npm install
```
- Запуск проекта
```
npm start
```

### Технологии:
- Typescript
- SCSS, CSS Modules
- React
- Redux
- Redux-thunk
- Ant Design
